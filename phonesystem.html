📋 Penjelasan Detail Alur Kerja Hardware

1

POWER BUTTON

Power Button ditekan. Sinyal interrupt GPIO ke PMIC. Membangunkan sistem dari deep sleep.

2

PMIC ACTIVATION

PMIC (PM8550) aktif. Regulator: VDD_CPU (0.5-1.0V), VDD_GPU, VDD_MEM (0.5V), VDD_IO (1.8V). Daya dari Baterai 3.85V.

3

SoC BOOT

SoC (Snapdragon 8 Gen 2) menerima VDD. Boot ROM (PBL) dieksekusi. PLL clock diinisialisasi.

4

MEMORY INIT

LPDDR5X RAM via bus DDR (64-bit, 4200MHz). UFS 4.0 Storage via M-PHY (23.2Gbps). Bootloader dimuat ke RAM.

5

DISPLAY & CAMERA

Display Driver IC via MIPI DSI (4-lane, 2.5Gbps/lane). ISP via MIPI CSI dari sensor kamera.

6

RF & CONNECTIVITY

X70 5G Modem (mmWave + sub-6GHz). WiFi 7, BT 5.3, GPS (L1+L5), NFC. Antenna tuner (QAT) optimasi.

7

SENSORS & HAPTIC

Sensor Hub: Accel (BMI270), Gyro, Mag, Prox (VCNL4040), ALS, Baro, Ultrasonic Fingerprint. LRA Vibrator aktif.

8

AUDIO SYSTEM

WCD9385 Audio Codec. Stereo speakers (1216 box), 4x MEMS microphones, headphone amp. Bus I2S/PCM.

9

CHARGING PATH

Charger USB-C: SMB1398 Charging IC atur arus (max 120W). USB PD 3.1 negosiasi 5V-20V. PMIC kontrol CC/CV + thermal protection.

Layout: CPU di Pusat | Komponen Mengelilingi | Power/Volume di Bawah dengan Gap

Gambar sesuai jenis: IC = chip package, Baterai = cell, USB-C = port, Button = side key
